There are a lot of techniques to hack Facebook like phishing ,bruteforce ,cookie stealing etc. Here, we will learn about bruteforce.

Bruteforce

It is the process of trying different combinations of characters as password and trying to login into an account. For this technique, first we need a wordlist file. A wordlist is a file with different password combinations. The hacking script takes each password, one at a time and tries it on an account. There are different types of brute force attacks viz. simple bruteforce and reverse bruteforce. Unlike simple bruteforce, reverse bruteforce uses a single password in multiple accounts.The tool that I use for hacking is
